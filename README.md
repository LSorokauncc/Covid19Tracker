## Features

- **Map with interactive heatmap overlay** (powered by Mapbo)
- **Dark Mode** (toggle UI theme)
- **Compare regions with drop down menu**
- **Search Bar** for country lookup
- **Expandable card in WIKI Page** 

## Navigation Pages
